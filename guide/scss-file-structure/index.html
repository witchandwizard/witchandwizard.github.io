SCSS File & Structure

New line’s template had been developed with modular code in mind. Therefore you will find multiple scss file inside scss folder. Each file is responsible for a section or a specific concern. All of them would be included inside the style.scss.

Skin style

There is no hardcoding inside our theme. We do our best to make sure each and every detail is configured in one place. Therefore you will find all New line’s option inside folder skin. We made a single skin call _grey.scss there.

Common rule style

All the common rule for our template are also divided to specific file. You can find those information in _common.scss file. Those common style include:

_button.scss

This file includes all the style related to button.

_animate.scss

This file includes all the style related to Css3 animition.

_flexslider.scss

This file includes all the style related to flexslider. All the rewrite style we need to adapt the flexslider for out template.

_parllax.scss

This file includes all the style related to parallax effect.

_prettyphoto.scss

This file includes all the style  related to prettyphoto that we have customized.

_section.scss

This file includes all the common style for section.

Also some other file is little specific and we keep it outside such as:

_rest.scss

This file is to rest the basic of font, color of most basic html tag

partial/_section_header.scss

This file includes style for all the section header.

_modal.scss

This file includes style for our theme’s modal.

Normal section file

Each section file below contains the style that control how that section look, the detail position of each element. The mobile and desktop style of each section all contained in same file.
