Javascript Structures

Even though we are based on Bootstrap to develop this template, we are also come a step further to provide a custom experience for every section.  Those custom experience is powered by our javascript’s structure.

This javascript’s structure does not simply a bunch of call inside of a $(document).ready but had been organized to systematic structure. This will allow you to quickly get use to it to perform any further required customization.

All the javascript of this template can be found in main.js inside js folder. The detail for each effect and section is as below:

Sticky and menu highlighted

The menu is stocked to the top of the page when user scroll to other section. This is support by our function initStickyMenu in layout  object. The main idea is to add support class navbar-fixed-top from bootstrap at suitable scrolltop value

Also we develop a custom piece of code inside this function also to highlighted the active menu.

Testimonial section

The slide effect in this template is supported by Flexslider. All the slide effect had been inited by the call of the fucntion initFlexslider of the layout object

Project modal box

To allow user to view the project in modal box style, we have included the support of pretty photo. The initializaiton code can be found in function initPortfolioGallery of layout object.

Parallax effect

To provide the parallax’s effect we have used in open source stellar plugin. All the option and initialization can be found in initParallax of layout object.

Scroll effect

In order to provide smooth and flexible scrolling effect we have include the scrollTo libary. All the code for this part can be found in function initClickScrollTo of layout object

Map

Map custom expand is covered by initMap function

Video section

Video trigger, Video responsive is customized inside initVideo function
