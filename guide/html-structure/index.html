HTML Structure

The HTML structure heavily based on HTML structure of the Bootstrap Front end template.

Our responsive HTML structure have been developed from 1170px grid layout of Bootstrap Framework. HTML structure for specific section can be found below.

 Header

New Line’s header html includes

  • A big background image that is represented by img tag.
  • Floating menu that is at the bottom of header and will be a sticky menu when user scroll to other section. The html for this floating menu is based on navbar of Bootstrap.
  • Position absolute content with h1 and h2 to hold the floating text. This text is wrapped in a container div of bootstrap to provide it sufficient margin.

Refer to header section in index.html for more information

Video

New Line’s html includes

  • Dark Full width background image to make a unique section appearance.
  • Contrast text and a play trigger icon to play video.
  • Video contained inside Bootstrap modal with the ability to be responsive.

Service section (Feature)

New Line’ service section developed based on bootstrap grid. The margin is provided by class container of Bootstrap. The detail implementation includes

  • Group of 3 column default item grid. Each of the item support
    • h3 tag for title
    • Icon using i tag and Elegant icon
    • p tag to hold the description

Refer to service section on index.html for more information

Project section

Akaposs’s skill section also use the same margin that had been provided with bootstrap container and row class. The HTML’s structure is compatible with pretty photo plugin. The details is as below:

  • Same header’s structure as other section.
  • Each project represented as a bootstrap column.
  • Each project item consist of:
    • div class title for holding the project’s title
    • Zoom icon from Elegant icon.
    • a tag with compatible html structure for pretty photo plugin

Refer to Project’ssection index.html for more information

Testimonial section

Newline’s testimonial section also use the same margin that had been provided with bootstrap container and row class. The HTML’s structure is compatible with flexslider plugin. The details is as below:

  • Flexslider’s html structure to prove slide function
  • Each flexslider slides consist of  html of single quote from customer.
  • Each item have the same html as follow:
    • div with “msg” class to hold the content of testimonial
    • div with “position” class to hold further information of the testimonial’s author.
  • A ul list of customer logo

Refer to Testimonial’ssection index.html for more information

Achievement section ( Business promote )

Newline’s achievement section currently developed to support an impression text message on an dark’s background. This section’s html structure includes:

  • Full width dark background.
  • Four column div with class center-text to hold the text message.
  • Separate line using dive with height 1px

Refer to Achievement section index.html for more information

Team section

Newline’s team section also use the same margin that had been provided with bootstrap container and row class. The details is as below:

  • Four column div each for team member.
  • Social anchor group using ul.
  • Simple job’s title and name in p text.

Pricing section

New line’s pricing section had been developed using container and row class of bootstrap to provide adequate margin support. The detail HTML structure included.

  • Three plans using ul.
  • Each plans consist of:
    • header class to hold the plan’s name and icon.
    • li list for main text
    • trigger button
  • Each plans icon is developed using Elegant icon

Refer to Pricing’s section index.html for more information

Blog section

New line’s blog section also use the same margin that had been provided with bootstrap container and row class. The HTML’s detail consist of:

  • Blog image using img tag.
  • separate div for blog’s writer and a date.
  • p tag to contains the blog’s description

Refer to Blog’s section index.html for more information

Contact section

Akaposs’s contact section also use the same margin that had been provided with bootstrap container and row class. This section included 3 separate area

  • Div left with
    • h4 title
    • p description
    • li compan’s information
  • Div right with form input and text area to submit information
  • Map area compatible to Google maps

Footer section

Akaposs’ footer section using a simple ul list to list the icon.

Refer to Footer’s section in index.html for more information
